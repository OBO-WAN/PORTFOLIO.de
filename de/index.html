Hier ist ein Überblick über meine Fähigkeiten. Ich halte mich gerne über die neuesten Technologien und Best Practices in der Frontend-Entwicklung auf dem Laufenden und lerne immer gerne Neues.

Ich bin motiviert, meine Fähigkeiten in laufenden Projekten einzubringen und suche neue Herausforderungen, um zu wachsen und mein Fachwissen zu erweitern.

1

Project Join

  • Über das Projekt

    Auf dem Kanban-System basierender Task-Manager. Aufgaben lassen sich per Drag-and-Drop erstellen und organisieren sowie Benutzern und Kategorien zuweisen.

  • Verwendete Technologien

    Angular, TypeScript, HTML, CSS, Firebase

  • Was ich gelernt habe

    Meine Erfahrung in Gruppenprojekten, Teamarbeit, Kommunikation und Planung während der Umsetzung.

GitHub Live-Demo

2

Project Pollo Loco

  • Über das Projekt

    Jump-and-Run-Spiel auf Basis eines objektorientierten Ansatzes. Hilf Pepe, Münzen und Tabasco-Salsa zu finden, um gegen das verrückte Huhn zu kämpfen.

  • Verwendete Technologien

    Angular, TypeScript, HTML, CSS, Firebase

  • Was ich gelernt habe

    Ich habe gelernt, wie man ein Spiel mit objektorientierten Programmierprinzipien umsetzt und dabei eine unterhaltsame Benutzererfahrung schafft.

GitHub Live-Demo

3

Projekt Memory

  • Über das Projekt

    Responsives Memory-Kartenspiel für zwei Personen mit wählbarer Kartenzahl, abwechselnden Spielzügen, Punktestand und animiertem Kartenfeedback.

  • Verwendete Technologien

    TypeScript, SCSS, Vite, HTML

  • Was ich gelernt habe

    Ich habe meine TypeScript-Kenntnisse vertieft, indem ich Spielzustand, Zuglogik und Kartenvergleiche strukturiert sowie eine responsive Oberfläche und ein wiederverwendbares Theme-System umgesetzt habe.

GitHub Live-Demo
